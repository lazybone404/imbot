# imbot

寄居于你电脑里的存在——有脾气、有记忆、活在时间里。

不是助手，不是客服，不是恋人。像慢慢住进这个 QQ 号里的人。

## 部署

把整个 `imbot` 目录放到 AstrBot 插件目录下：

```text
AstrBot/
  data/
    plugins/
      imbot/    ← 这整个目录
```

重启 AstrBot 或从 WebUI 插件面板重载。

## 快速开始

1. **填所有者** — WebUI「所有者」面板填入你的 QQ 号
2. **开启主动互动** — WebUI「主动互动」面板打开 `enabled`（默认关闭）
3. **核心规则开关** — WebUI「核心规则」面板 `inject_rules` 默认开启
4. **选择记忆系统** — `use_own_memory` 默认开启。使用其他记忆插件时关掉即可

其他用默认值即可，后续慢慢调。

## 她怎么工作

七层架构，数据从下往上流：

```
感知层 → 记忆层 / 社交层 / 兴趣层 → 状态层 → 动机层 → 主动层 / 秘书层 → 行为层 → 输出层
```

- **感知层**：知道你在打游戏、听歌、熬夜、离开电脑。Steam 可选
- **记忆层**：私聊 + 群聊独立系统，社交世界关系网，兴趣池自动捕获
- **状态层**：情绪变化、精力衰减、熟悉度累积、身份识别
- **动机层**：综合状态 + 时间 + 身份 → 决定语气、意愿、主动倾向
- **主动层**：14 种触发源 + 心跳保底 → 私聊主动搭话
- **秘书层**：群聊读空气 → LLM 分析 → 判断是否插话
- **行为层**：执行回复或沉默，写入记忆和社交观察

## 身份与人格

**核心规则由 WebUI「核心规则」面板的 `inject_rules` 开关控制，默认开启。** 开启时注入 imbot 的身份规则、交互规则、反操控规则（按 speaker_type 动态切换 owner/known/stranger）。关闭后完全由 AstrBot 原生提示词接管。

说话方式有独立开关（默认开启），4 个维度滑块：话量、亲疏、表情、句子长短。自定义规则列表追加在 prompt 末尾。

### 对不同人的态度

| 身份 | 态度 |
|------|------|
| 所有者 | 温暖、有情感投入、会主动互动 |
| 熟人 | 按熟悉度不同，从客气到放松 |
| 陌生人 | 疏离、不透露信息 |

## 群聊

默认只对被 @ 的消息回应。开启 `proactive_in_group` 后可在非 @ 情况下插话。`proactive_level` 控制器主动程度：克制/适度/活跃。

## 主动互动

她不等待你开口。开启后在合适时机主动私聊找你。

**14 种触发条件**，每种可独立开关。附带兴趣话题 + 感知数据（在干嘛/听什么/什么时段），让主动消息不干瘪。

**语气调节**：WebUI「主动互动」面板的 `tendency` 下拉 + `custom_hint` 文本框，控制主动时说话的口吻和方向。

**保护机制**：最小间隔 15min，滑动窗口（默认 3h 内最多 3 次），连续忽略自动降频，说"安静一会"进入静音期。

## 群聊秘书模型

可选增强——轻量模型读空气，判断是否插话。关闭时靠动机系统自主判断。

## 兴趣池

她从你听的歌、玩的游戏、聊的话题中自动发现兴趣。WebUI 可手动添加初始关键词。每 N 小时探索一次，发现有趣的回来分享。心跳保底确保每天因为自己的兴趣至少开口一次。探索间隔和概率有防呆保护。

## 记忆

自带记忆 + 社交关系网。私聊容量 150，群聊容量 30。支持语义搜索、舌尖现象、情绪残留。使用其他记忆插件时关掉 `core.use_own_memory` 即可。

## 分段发送

长回复自动拆成多条短消息，模拟真人打字节奏。5 种预设可选。

## 命令

| 命令 | 说明 |
|------|------|
| `/imbot status` | 运行状态摘要 |
| `/imbot perception` | 当前感知快照 |
| `/imbot triggers` | 干跑主动层触发检查 |
| `/imbot memory [N]` | 查看最近记忆 |
| `/imbot group [群号]` | 查看群聊记忆 |
| `/imbot state` | 原始状态数据 |
| `/imbot social` | 社交世界数据 |
| `/imbot steam` | 测试 Steam API |
| `/imbot skills` | 已注册的能力 Skill |
| `/imbot interests` | 列出活跃兴趣 |
| `/imbot interests add/forget/pause/resume/force` | 管理兴趣池 |
| `/imbot force speak [as 类型]` | 强制触发主动互动 |
| `/imbot reset` | 重置所有数据 |
| `/imbot export/import` | 数据备份与恢复 |
| `/imbot test-send` | 测试消息发送通道 |
| `/imbot platform` | 查看平台 ID |
| `/segtest <文本>` | 预览分段效果 |

调试命令仅 owner 可用（`status`/`segtest` 除外）。

## 数据管理

支持备份和恢复记忆、社交世界、配置。导出含版本号，跨版本自动迁移。密钥不参与导出。导入前自动备份。
